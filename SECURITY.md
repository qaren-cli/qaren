# Security Policy

## Supported Versions

Only the latest stable version of Qaren is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

We take the security of Qaren seriously. If you believe you have found a security vulnerability, please do NOT report it via a public issue.

Instead, please send an email to the maintainers (or use GitHub Private Vulnerability Reporting if enabled).

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
